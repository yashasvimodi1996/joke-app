A fun web application that delivers jokes with an animated punchline reveal and categorized joke collections.

## Features

- Beautiful dark-themed UI with gradient text
- Animated punchline reveal with smooth transitions
- **Multiple joke categories:**
  - Adult humor
  - Sports jokes
  - Science jokes
  - Fashion jokes
  - Trivia jokes
  - Politics jokes
  - All (mixed collection)
- Category filtering with active state visual feedback
- Joke counter showing current position in selected category
- Responsive design that works on all devices
- Get a new joke with a single click

## How to Use

1. Open `index.html` in your web browser
2. Select a joke category using the category filter buttons at the top
3. Click "Reveal" to show the punchline with a smooth animation
4. Click "Next Joke" to see another joke from the selected category
5. Switch between categories anytime to explore different joke types

## Categories

- **All** - Mixed collection of all jokes (10+ jokes)
- **Adult** - Adult humor jokes (5+ jokes)
- **Sports** - Sports-related jokes (5+ jokes)
- **Science** - Science and math jokes (5+ jokes)
- **Fashion** - Fashion industry jokes (5+ jokes)
- **Trivia** - General trivia jokes (5+ jokes)
- **Politics** - Political humor (5+ jokes)

## Technologies

- HTML5
- CSS3 (with animations, gradients, and flexbox layouts)
- Vanilla JavaScript (no dependencies)
