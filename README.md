A fun web application that delivers jokes with an animated punchline reveal.

## Features

- Beautiful dark-themed UI with gradient text
- Animated punchline reveal with smooth transitions
- Get random jokes with a single click
- Responsive design that works on all devices

## How to Use

1. Open `index.html` in your web browser
2. Click "Get Joke" to fetch a new joke
3. The setup appears first, and the punchline reveals with a smooth animation
4. Click "Get Joke" again to see another joke

## Technologies

- HTML5
- CSS3 (with animations and gradients)
- JavaScript
- Joke API (for fetching random jokes)
